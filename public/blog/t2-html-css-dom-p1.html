Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is all the text on a page. It is like a supermarket and all the produce inside it. CSS is like the layout of the supermarket. All the aisles and signs making it easier to navigate. The DOM is a representation of that supermarket to make it easier to interact with. For example, shopping online you don't pick up every item to inspect it, you can only choose the kind of item to buy.

What is meant by boxifying design?

When you look at a webpage you can see that everything appears to be laid out in segments according to its contents and function. This is boxification.

What is the box model?

The box model is how CSS places the content on a page or within an element. It specifies the height and width of the content, its padding, its border and its margin.
