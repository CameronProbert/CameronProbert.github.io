Scope

Scope defines where functions and variables (data) in Javascript can be used. If a variable is defined without using the var keyword, or it is defined outside any brackets and before any code it will have global scope. This means that any code in the Javascript element can use the variable.

If it is not global then only code within the same nested level or a child of the nested level can access it. This has the benefit of being able to reuse variable names when you want to use them for common functionality, like accessing the index of an array with variable 'i', but doesn't risk having any concurrency issues due to multiple methods accessing the same global variable at the same time.
