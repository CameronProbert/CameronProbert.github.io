How does JavaScript compare to HTML and CSS?

Javascript is used alongside HTML and CSS to alter the page in response to the user's actions.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the process of doing something, for example, when brushing your teeth you:

  1. Grab your toothbrush and toothpaste.
  2. Put toothpaste on your toothbrush.
  3. Wet your toothbrush.
  4. Brush your teeth.
  5. Spit the toothpaste out.
  6. Brush your tongue.
  7. Rinse your toothbrush.
A loop is when you repeat this, such as in the morning and evening. (And after lunch in South Korea where I'm teaching at the moment!)

Explain the difference between accessing data from arrays and objects.

When accessing data from an array you need to know where in the array the data is and use a number to refer to that, such as the 5th object in the array. When accessing data from an object you need to use the name given to that piece of data, for example, object.eyeColour.

Explain what functions are and why they are useful.

Functions are sections of code that has been given a name. You can make the code run by typing the name. This is very useful to avoid having to retype code all the time, making the code more readable and easier to alter.
