What is grid based design?

Grid based design is a way of boxifying the layout of a web page in a logical manner. It helps the developer to visualise the layout as they are coding, and to easily modify the layout of the elements.

Whats all the hype about responsive webpages?

There are so many different sized screens being used to access webpages nowadays that a webpage is not going to look attractive on each webpage unless the webpage responds to the screen size.

What is semantic structure?

Semantic structure is structure (such as HTML tags) that makes it easier for humans to understand the code.
