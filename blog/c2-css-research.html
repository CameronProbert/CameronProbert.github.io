'Screen Filler' by Elliott Jay Stocks

I like that this design utilises the whole screen. I think the colour palette, while strong, doesn't clash with itself. I think that the transparent containing divs look good, and would like to implement that myself.

'A Robot Named Jimmy' by Meltmedia

I like the alternating left and right aligned images that this css implements. I have decided for the moment to implement this into my website, using the flex layout.

'Verde Moderna' by Dave Shea

I like the full page look of Verde Moderna. The subtle design behind the content of the page looks really nice. I think that seeing this design, as well as Screen Filler (above), I can see that slight transparency can really make a website pop.

'Garments' by Dan Mall

The simplistic design of this css caught my attention. I would consider trying to implement in my blog simple icons as shown here. In Verde Moderna instead of stylised icons like this, abbreviations were used as links to these other pages, but I couldn't tell what they were linking to without hovering over them to see the address context tips.
